Initial release of AIM-ALPHA documentation.

\begin{changelogentry}{2026-2-11}{R. Totake}{1.1.0}
Added protected areas; split forests into primary and secondary categories; and divided pastureland into grazed and ungrazed categories to more accurately represent land use and grazing intensity.
\end{changelogentry}
\begin{changelogentry}{2026-2-21}{R. Totake}{1.1.1}
Added SSP-based urban land change input (SEDAC 1-km urban dataset) with LUH2-consistent base-year handling
\end{changelogentry}
\begin{changelogentry}{2026-2-27}{R. Totake}{1.2.0}
Updated scenario and inter-model integration settings: introduced split scenario controls (\texttt{socioeconomic\_setting}, \texttt{climatepolicy\_setting}, \texttt{climate\_setting}), standardized scenario suffix control with \texttt{ModelInt}, changed \texttt{runlist} format to \texttt{socioeconomic|climatepolicy|climate}, and added optional pre-run bridge processes (\texttt{hubauto}/Hub2ALPHA and \texttt{afoluauto}/AFOLU2ALPHA).
\end{changelogentry}
